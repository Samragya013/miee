# MIIE Examples

Runnable examples demonstrating MIIE functionality.

## Examples

| Example | Description |
|---|---|
| [analyze_local.py](analyze_local.py) | Analyze a local repository |
| [analyze_github.py](analyze_github.py) | Analyze a GitHub repository |
| [export_markdown.py](export_markdown.py) | Export results as Markdown |
| [export_json.py](export_json.py) | Export results as JSON |
| [cli_usage.py](cli_usage.py) | Interactive CLI usage |
| [python_api.py](python_api.py) | Python API usage |
| [batch_analysis.py](batch_analysis.py) | Batch analysis of multiple repos |

## Running Examples

```bash
# Run any example
python examples/analyze_local.py

# With arguments
python examples/analyze_local.py /path/to/repo
```
